# ai-speaking-tutor
This project is an AI-driven system designed to support both learners and recruiters by combining personalized tutoring with candidate screening capabilities. It uses NLP concepts and agent-based AI approaches to create a spoken interaction system that adapts to user needs.

The system helps learners receive personalized guidance while enabling HR teams to perform faster and more consistent early-stage candidate screening based on qualification profiling.

Tech Stack: Python, Agentic AI, NLP, Microservices
